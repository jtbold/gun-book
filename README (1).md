# Met Museum Firearms Collector

A web tool to search and download public domain firearm images from the Metropolitan Museum of Art's Open Access collection.

## Features

- **Era Selection** - Filter by time period (Pre-1600 through 20th Century)
- **Search Terms** - Toggle specific firearm types (double barrel shotguns, rifles, fowling pieces, etc.)
- **Exclusion Filters** - Remove unwanted items (pistols, revolvers, etc.)
- **Download Scripts** - Generate PowerShell (Windows) or Bash (Mac/Linux) scripts
- **Image Preview** - Click any image to view full size

## Usage

1. Visit the live site
2. Select an era
3. Toggle search terms (green = included)
4. Toggle exclusions (red = excluded)
5. Enter your desired save folder path
6. Click "Search Met Museum"
7. Generate a download script and run it locally
